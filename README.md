# AI Email Spam Detector

A machine learning-powered email spam detection system built using Python, NLP, Scikit-learn, and Streamlit.

## Features
- Spam vs Not Spam classification
- Confidence score prediction
- Spam keyword highlighting
- Interactive Streamlit dashboard
- Real-time email analysis

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
- NumPy
- NLP (TF-IDF)

## Dataset
- SMS Spam Collection Dataset
- 5572 labeled messages

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
